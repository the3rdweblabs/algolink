# Algorand Wallets

## Pera Wallet

Pera Wallet is widely regarded as the best and most popular wallet for the Algorand blockchain. It was originally the official Algorand Wallet (from the Algorand Foundation), but got rebranded to Pera Wallet a while back. Key features and details:

- Self-custodial — you control your keys completely (secure and decentralized).
- Supports storing, sending, receiving ALGO + other Algorand assets (ASAs), NFTs, etc.
- Built-in swaps, staking, buying crypto with fiat (credit card), and dApp connections.
- Available as a mobile app (iOS + Android), plus a web version at web.perawallet.app.
- Open-source, community-driven, and frequently updated.
- Active communities on Discord (discord.gg/gR2UdkCTXQ), Telegram, and Reddit.

## Lute Wallet

Lute is a popular web-based cryptocurrency wallet built specifically for the Algorand blockchain (and AVM – Algorand Virtual Machine). Key details about Lute Wallet:

- It's a web wallet (accessible via browser at lute.app) and also available as a Chrome extension.
- Originally designed with strong Ledger hardware wallet support (great for secure, hardware-backed signing without relying on WalletConnect issues that plague some other wallets).
- It allows you to manage Algorand accounts, connect to dApps, sign transactions, handle rekeyed accounts, switch networks easily, support multiple accounts per Ledger, and more advanced features like connecting to your own node or "Snoop Mode".

Links:
- Lute Wallet Website: https://lute.app/
- Lute Wallet X Account: https://x.com/LuteWallet
- GitHub (Lute Connect): https://github.com/GalaxyPay/lute-connect

## Defly Wallet

Defly is a popular mobile wallet built specifically for the Algorand blockchain, with a strong focus on DeFi (decentralized finance) and trading. It's often called Algorand's first DeFi wallet or the "DeFi-optimized" wallet, designed to make decentralized trading feel more like centralized exchanges while keeping full self-custody and security. Key features and details:

- Self-custodial non-custodial wallet — you fully control your keys and funds (stored securely on your device).
- Advanced trading tools: In-app swaps (optimized across multiple DEXs like Tinyman, Pact, Humble for best rates), limit orders, real-time charts, and market data aggregation.
- Portfolio tracking: Monitors your DeFi positions (staking, farming, lending, governance commitments across protocols like Folks Finance, Algofi, etc.).
- Supports multisig, Ledger hardware integration, dApp browser, and staking of its own token $DEFLY in-app.

Links:

- Defly X account: https://x.com/deflyapp
- Defly Docs: https://defly.gitbook.io/defly-manual/
