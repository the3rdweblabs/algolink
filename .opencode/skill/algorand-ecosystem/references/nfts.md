# NFTs (Marketplaces & Tools)

## Downbad

Downbad is a prominent NFT marketplace built on the Algorand blockchain. It serves as one of the leading platforms for discovering, buying, selling, and trading NFTs specifically on Algorand.

Key features include:
- Browsing featured collections and trending assets.
- Viewing recent listings, sales, and activity.
- Unique tools like Shuffles
- Staking options for NFTs (via staking.downbad.farm)
- Portfolio tracking, collections explorer, and user profiles.

Links:
- Downbad Website: https://www.downbad.farm
- Downbad Marketplace: market.downbad.farm

## Rand Gallery

Rand Gallery is the leading and primary NFT marketplace built specifically for the Algorand blockchain. It's often described as "The Home of Algorand NFTs" and serves as a platform where users can create (mint), buy, sell, list, and explore non-fungible tokens (NFTs) on Algorand.

Key points about Rand Gallery:
- It focuses exclusively on Algorand-based NFTs
- Longest-running NFT Marketplace; recently acquired by the Goanna DAO
- Users connect Algorand wallets to mint NFTs directly on the blockchain and trade them with low fees and fast transactions, leveraging Algorand's strengths.
- The platform includes features like browsing trending collections, recent listings/sales, and activity feeds

- Rand Gallery Website: https://www.randgallery.com/

## Wen Tools

Wen.tools is a suite of free and open-source, no-code tools built specifically for the Algorand blockchain. It makes it easier for creators, collectors, developers, and everyday users to handle common tasks like minting, sending, and managing assets without writing code or dealing with complex setups. It's often praised in the Algorand community (including mentions from Pera Wallet folks and ecosystem overviews) as a go-to for simplifying bulk operations and onboarding people into NFTs, tokens, and airdrops on Algorand.

Key features and tools include:
- **Simple Mint / Really Simple Mint / Simple Mint Classic** — Easy ways to mint single or bulk NFTs and fungible tokens using standards like ARC-69, with decentralized IPFS pinning via partners like Crust Network (you pay fees in ALGO, and it's more decentralized than many other platforms).
- **Simple Send** — Mass send assets (e.g., one token to many wallets, or multiple assets to one wallet).
- **Airdrop** — Tool to airdrop tokens, assets, or ALGO to lists of addresses or holders.
- **Batch Opt-in (Asset Add)** — Bulk opt-in to assets so you can receive them without spam issues.
- **Vault Send** — Send to NFD-controlled vaults (which auto-opt-in).
- **Multimint Asset Holders** — Export holder lists for multimint assets in CSV.
- **Token Manager** — Manage circulation supply for tokens using ARC-62 standards.

- Wen Tools Website: https://www.wen.tools/

## Minthol

Minthol is a browser/explorer and analytics tool for NFTs (non-fungible tokens) built on the Algorand blockchain. It aggregates and displays data across various Algorand-based NFT collections, allowing users to browse, view, and analyze NFTs without necessarily being a full marketplace for buying/selling (though it links into the ecosystem).

- Minthol Website: https://www.minthol.art/

## NFDomains

NFDomains (often stylized as NFD or nf.domains) is Algorand's native name service and marketplace for Non-Fungible Domains (NFDs). These are essentially Web3 domain names ending in .algo — unique, human-readable identities tied to your Algorand wallet address. Think of it like ENS (.eth) on Ethereum or similar services on other chains, but built specifically for Algorand.

- NFDomains Website: https://app.nf.domains/
- NFDomains API: https://api-docs.nf.domains/
- NFDomains SDK: https://github.com/TxnLab/nfd-sdk

## GoPlausible

GoPlausible is an identity, credentialing, and AI-agent infrastructure provider built on Algorand, offering a full W3C-compliant stack for Decentralized Identifiers (DIDs), Verifiable Credentials (VCs), OpenBadges, and smart utility NFTs. It also extends into agentic AI, enabling AI systems to interact with Algorand through the Model Context Protocol (MCP). GoPlausible provides an x402 facilitator for API/MCP/A2A access.

- GoPlausible Website: https://goplausible.com/
- OpenClaw Algorand Plugin: https://npmjs.com/package/@goplausible/openclaw-algorand-plugin
- Algorand x402: https://x402.goplausible.xyz/
- Algorand x402 Facilitator: https://facilitator.goplausible.xyz/
- Algorand x402 test endpoints: https://example.x402.goplausible.xyz/
